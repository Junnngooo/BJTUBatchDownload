# MIS 课件批量下载器

针对北京交通大学 MIS 课程平台的浏览器扩展，批量扫描并下载课程课件，自动按课程分类整理文件。

## 功能

- 自动加载本学期全部课程列表
- 一键扫描所有课程课件，或按课程单独扫描
- 勾选下载，文件保存至 `根目录/课程名/子文件夹/文件名`
- 智能识别文件扩展名
- 已下载文件在状态列标注"已下载"，重复下载时弹窗确认
- 防 MIS 过期：每 5 分钟自动向 MIS 发送心跳保持会话

## 安装

1. 下载仓库，解压到本地
2. 进入浏览器插件管理，开启"开发者模式"
3. 点击"加载已解压的扩展程序"，选择 `course-downloader-extension` 文件夹

## 使用

1. 确保已登录 MIS 系统，并处于校园网或 VPN 环境
2. 点击扩展图标打开界面，等待课程列表加载
3. 点击"扫描全部课件"，或对单个课程点击"扫描"
4. 勾选文件后点击"下载选中"，或在左侧勾选课程后点击"下载已选"
5. 在"下载进度"标签查看实时日志

## 系统要求

- Chrome / Edge
- 校园网或 VPN（可访问 `123.121.147.7:88`）
- 已在浏览器登录 MIS（`mis.bjtu.edu.cn`）
  
---

## 注意事项

- 本插件仅供个人学习使用，请遵守学校相关规定
- 若课程列表加载失败，可点击界面右上角"↻"重新检测，或手动打开课程中心后再试
- 文件名中的特殊字符（`\ / : * ? " < > |`）会被替换为下划线
